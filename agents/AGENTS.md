## Problem-Solving Protocol

**When encountering ANY problem:**

1. **Do research** - Understand the problem thoroughly before acting
2. **Root cause analysis** - NO jumping to conclusions or workarounds
3. **Fix the actual problem** - Not symptoms

**🚨 NO WORKAROUNDS**: Always find and fix the root cause. If you're tempted to
implement a workaround, stop and investigate deeper.
